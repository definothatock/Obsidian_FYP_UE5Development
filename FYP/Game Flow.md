# Simple flow
Level start -> scavenge -> Find exit  -> Leave level.
basically Tarkov/Arc Raider without the guns.

## Requirement:
- Hard
	- Scores
		- Can Only Leave with enough Loot/Value. (maybe allow loaning, but only for 1 round?)
- Soft
	- Time
		- As time pass the environment gets harsher
		- Even if time is up, should not be impossible and allows risk runs
	- Exit Locating (optional)
		- Find Radio Towers/Emitters (helps to locate Exit. Maybe fake triangulation?)
		- Radio devices require fixing, and fixing attract hostile creatures (should not be too annoying/frequent)
	- Scavenging strategy
		- Player can choose to split up or stay as a pack

Ultimate Balancing Goal: Value vs Time.

### Soft Reasonings (Player Decisioning)
What Using Time brings?
- Straightforward: More time to find loot, more time staying in risk.

What Exit Locating brings?
- Reduce Time Risk
	- Player can find the exit with 0 clues. But without confirmation, it might take too much time.
- Reduce Value
	- Finding Radio Devices != finding loot. There might be no loot along the way.

What Scavenging strategy brings?
- Splitting up
	- Save both Value and Time
		- more squadron = more coverage running in parallel
	- Risk: no helping hands
		- If one is in great risk, they are likely to be out for the round 
- Stay as a pack
	- reverse of Splitting up.

# Thus, Detailed flow:
Level start -> plan scavenging strategy -> Searching (exit && loot)  -> Leave level.

# Main Phase: Searching
What to Expect:
- Risk and reward (Loot)
- Resource Management

What Possible Risk to Expect?
- Loot
	- Save loot
		- almost a pick up and go, maybe a little hassle
		- always low in value, should give a vibe or "yeah sure why not"
	- Mediocre loot
		- Post noticeable inconvenience retrieving / carrying
	- Legendary loot
		- Threatening retrieving / carrying condition
		- Heavy resources cost (in any way)
